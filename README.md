# The Digital Janitor - Landing Page

A stunning, high-conversion landing page for The Digital Janitor desktop app.

## Tech Stack

- **Next.js 14** - React framework with App Router
- **Tailwind CSS** - Utility-first CSS framework
- **Framer Motion** - Animation library
- **Lucide React** - Icon library

## Getting Started

1. Install dependencies:

```bash
npm install
```

2. Run the development server:

```bash
npm run dev
```

3. Open [http://localhost:3000](http://localhost:3000) in your browser.

## Building for Production

```bash
npm run build
```

The static output will be in the `out` directory, ready for deployment to any static hosting service.

## Design Features

- **Dark mode aesthetic** - Pure black background with glowing cyan accents
- **Apple-like minimalism** - Clean typography, generous whitespace
- **Glassmorphism cards** - Frosted glass effect with subtle borders
- **Smooth animations** - Scroll-triggered fade-in effects
- **Responsive design** - Optimized for all screen sizes
